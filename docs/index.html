Team Overview

Abstract

Our team explored different numerical schemes to solve the Navier-Stokes equations for 2D smoke dynamics simulation. Traditional numerical solutions to the Navier Stokes equations evaluate the advection, diffusion, force, and pressure operators using implicit methods. We first re-implemented a naive and exact version of the classical solution using Pandas3D. We then extended this approach by using explicit Euler's and Runge Kutta integration for the advection term. Finally, we analyze each numerical scheme's properties with a qualitative set of images.
